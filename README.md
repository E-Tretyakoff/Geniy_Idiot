# Geniy Idiot

Консольное приложение на C#, которое определяет в шуточном контексте “уровень интеллекта” пользователя через простые вопросы.

Сделал этот проект, чтобы потренироваться в:
- работе с консольными приложениями
- базовой логике и обработке ввода
- работе с файлами
- структурировании кода

---

## 🎯 Что это

Программа задаёт пользователю несколько вопросов, проверяет ответы  
и в конце выдаёт результат (например: гений, норм, идиот и т.д.).

По сути — простая викторина с логикой оценки.

---

## 🎮 Что реализовано

- Набор вопросов и ответов
- Проверка правильности ответов
- Подсчёт результата
- Вывод “уровня интеллекта”
- работа с файлами для хранения данных результатов и вопросов JSON

---

## 🧠 Как это устроено

- Основная логика — обработка вопросов и проверка ответов
- Данные (вопросы/ответы) могут храниться:
  - либо в коде
  - либо во внешнем файле

Программа работает в консоли и обрабатывает ввод пользователя пошагово.
