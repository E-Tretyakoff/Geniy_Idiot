# Шуточная игра "Гений-идиот"

## Что есть в игре

### Ввод имени
пользователь вводит имя для дальнейшего сохранения в турнирную таблицу
### Таймер
У пользователя есть 10 секунд что бы ответить на вопрос
### Турнирная таблица
Можно посмотреть турнирную таблицу, и удалить определённого пользователя
### Рестарт
При рестарте игра перезапускается (результаты сбрасываются и не сохраняются)
### Конец игры
Выводит результат игры "диганоз пользователя😂" и сохраняет в файл JSON
### Admin панель
В консольном приложении можно удалять и добавлять вопросы
## Техническая часть
* Игра реализована в двух форматах в консольном и winforms
* Был выведен отдельно проект .Common для того что бы две игры ссылались на работу с пользователями и вопросами
* Консольное приложение и оконное приложение друг о друге не знает, и по разному интропретирует удаление и добавление пользователей или вопросов
* В классах "Question_storage" и "User_Storage" реализована серелизация и деселиризация с файлами JSON
* Реализован класс работы с файловой системой
* Реализованны проверки на ввод чисел и на ввод большого числа через try catch
